Definition

What exactly is a harness?

A coding agent is built in three layers. You control the outermost one.

Your layer
User Harness — AGENTS.md, lint configs, git hooks, CI pipeline, architecture docs. What you build.
Vendor
Builder Harness — system prompt, tool-use loop, context management. Built by Anthropic, Cursor, OpenAI.
Lab
Model — the LLM itself. GPT-4, Claude, Gemini. You can't touch this.
The Framework — Martin Fowler / Birgitta Böckeler

Guides & Sensors.
You need both.

Guides tell the agent what to do before it acts. Sensors catch problems after it acts and let it self-correct.

FEED-FORWARD

Guides

  • AGENTS.md
  • CLAUDE.md
  • .cursor/rules/
  • architecture.md
  • coding conventions
  • bootstrap instructions
FEEDBACK

Sensors

  • pre-commit hooks
  • linter (Biome / Ruff / golangci-lint)
  • type checker
  • test suite
  • CI pipeline
  • code review

Guides without sensors → the agent repeats the same mistakes.
Sensors without guides → the agent doesn't know what "right" is.
